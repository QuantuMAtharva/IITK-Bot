Commands

Beginning with some Utility Commands

1. =help : Displays Help Message; =help command_name : Gives a brief info of a command
2. =iitk : Gives important official websites related to IITK
3. =snt : Gives Server links for all SnT Clubs, with few other details
4. =mnc : Gives links related to MnC Clubs
5. =acads : Gives important links related to Academic things, like Grade Calculator Website, Course Resorces Links etc

The IITK Email Verification System

This enables you to verify whether the person joining your server is from IITK or not (the email domain can even be changed to any domain).

Use =vstatus to get a list of things you can do

Use =verify so that the Bot will DM the corresponding user who used the command, where he/she has to message the required domain's email and send the verification code received on it


Beginning with the commands that require Admin permissions

1. =domainadd domain : Add a new domain which you want users to verify themselves with. Eg: =domainadd iitk.ac.in to make users verify themselves with IITK email
2. =enableonjoin : Enables DMing the new users joining the server to verify their email. [By Default it is False]
3. =disableonjoin : Disables the verification of new users joining the server
4. =domainremove domain : Removes a domain from the list of domains
5. =rolechange new_role_name : Change default role which the bot creates for verified users. [Default: IITK_Verified]
